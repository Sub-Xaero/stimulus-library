## Purpose

Toggle a password input between `type="password"` and `type="text"` to allow the user to see what they have typed.


## [Actions](https://stimulus.hotwire.dev/reference/actions)

| Action   | Purpose                                              |
|----------|------------------------------------------------------|
| `peak`   | Turn the `password` input into a `text` input        |
| `hide`   | Turn the `text` input into a `password` input        |
| `toggle` | Toggle the input between `password` and `text` types |

## [Targets](https://stimulus.hotwire.dev/reference/targets)

| Target     | Purpose                                | Default |
|------------|----------------------------------------|---------|
| `password` | The password input to allow peeking on | -       |

## How to Use

Within a wrapper that loads the controller, add an `input[type="password"]` with a `data-password-peek-target='password'`
and an adjacent button/link with a `data-action` to call `toggle`, `peak`, or `hide`.

<StimulusPlayground
  controller="PasswordPeekController"
  html={`<div data-controller="password-peek"
  style="display:flex;flex-direction:column;gap:0.75rem;max-width:320px;">
  <label for="pwd">Password</label>
  <div style="display:flex;gap:0.5rem;">
    <input id="pwd" type="password" data-password-peek-target="password"
      value="s3cr3t-p4ssw0rd"
      style="flex:1;padding:0.5rem;border:1px solid #d1d5db;border-radius:6px;" />
    <button data-action="click->password-peek#toggle"
      style="padding:0.5rem 0.75rem;border:1px solid #d1d5db;border-radius:6px;cursor:pointer;">
      👁
    </button>
  </div>
  <small style="color:#6b7280;">Click the eye icon to reveal/hide the password.</small>
</div>`}
  editorHeight={220}
/>
