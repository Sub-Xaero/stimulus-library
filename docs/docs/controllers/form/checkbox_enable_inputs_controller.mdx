## Purpose

Enable other inputs if a checkbox is ticked.


## [Actions](https://stimulus.hotwire.dev/reference/actions)
<NoActions/>

## [Targets](https://stimulus.hotwire.dev/reference/targets)


| Target    | Purpose                                                                                            | Default |
|-----------|----------------------------------------------------------------------------------------------------|---------|
| `enabler` | The checkbox that when checked will enable other inputs, and disable them when it is *not* checked | -       |
| `enable`  | The input(s) that will be enabled when the `enablerTarget` is checked                              | -       |

## [Classes](https://stimulus.hotwire.dev/reference/classes)
<NoClasses/>

## [Values](https://stimulus.hotwire.dev/reference/values)

| Value   | Type    | Description                                                                       | Default |
|---------|---------|-----------------------------------------------------------------------------------|---------|
| `clear` | Boolean | Whether the controller should clear the targeted inputs when they become disabled | false   |

<NoEvents/>

<NoSideEffects/>


## How to Use

<Sandpack
  template="vanilla"
  files={{
    "/index.html": `<div class="container p-4 text-center">
  <div data-controller="checkbox-enable-inputs">
    <div class="row">
      <label class="form-check-label">
        I have read and agree to the <a href="#">terms and conditions</a>
        <input
          class="form-check-input"
          type="checkbox"
          data-checkbox-enable-inputs-target="enabler"
          data-action="change->checkbox-enable-inputs#toggle"
        />
      </label>
    </div>
    <br>
    <div class="row">
      <input
        type="submit"
        class="btn btn-primary"
        data-checkbox-enable-inputs-target="enable"
      />
    </div>
  </div>
</div>`
  }}
  customSetup={{
    dependencies: {
      "@hotwired/stimulus": "3.2.2",
      "@stimulus-library/controllers": "latest"
    }
  }}
  theme="light"
/>
