## Purpose

For elements that demand the user's attention, scroll an element into the view as it appears.

<Ephemeral/>

## [Values](https://stimulus.hotwire.dev/reference/values)

See [scrollIntoView documentation](https://developer.mozilla.org/en-US/docs/Web/API/Element/scrollIntoView) for possible values.

**Note**: If the user's browser does not support smooth scrolling, the browser will ignore these values and scroll using the browser default.

| Value       | Type   | Description                                | Default   |
|-------------|--------|--------------------------------------------|-----------|
| `behaviour` | String | One of `auto`, `smooth`                    | `auto`    |
| `block`     | String | One of `center`, `end`, `nearest`, `start` | `start`   |
| `inline`    | String | One of `center`, `end`, `nearest`, `start` | `nearest` |

  html={`<div style="max-width:420px;">

  <div id="item-list" style="height:200px;overflow-y:auto;border:1px solid #e5e7eb;border-radius:8px;padding:0.75rem;display:flex;flex-direction:column;gap:0.5rem;">

    <div style="padding:0.5rem 0.75rem;background:#f9fafb;border-radius:6px;font-size:0.9rem;color:#6b7280;">Item 2</div>
    <div style="padding:0.5rem 0.75rem;background:#f9fafb;border-radius:6px;font-size:0.9rem;color:#6b7280;">Item 3</div>
    <div style="padding:0.5rem 0.75rem;background:#f9fafb;border-radius:6px;font-size:0.9rem;color:#6b7280;">Item 4</div>
    <div style="padding:0.5rem 0.75rem;background:#f9fafb;border-radius:6px;font-size:0.9rem;color:#6b7280;">Item 5</div>
  </div>
  <button id="add-item"
    style="margin-top:0.75rem;padding:0.5rem 1rem;background:#6366f1;color:#fff;border:none;border-radius:6px;cursor:pointer;font-size:0.9rem;">
    + Add item (scrolls into view)
  </button>
</div>`}
  js={`let count = 5;
document.getElementById('add-item').addEventListener('click', () => {
  count++;
  const div = document.createElement('div');
  div.setAttribute('data-controller', 'scroll-into-focus');
  div.setAttribute('data-scroll-into-focus-behavior-value', 'smooth');
  div.style.cssText = 'padding:0.5rem 0.75rem;background:#eef2ff;border:1px solid #c7d2fe;border-radius:6px;font-size:0.9rem;color:#4338ca;';
  div.textContent = 'New Item ' + count + ' (scrolled into view)';
  document.getElementById('item-list').appendChild(div);
});`}
