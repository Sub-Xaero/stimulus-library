## Purpose

A utility controller to wire up other stimulus actions when the user is active on a page, and when the page gets minimised/loses focus.

## Events

| Event               | When                                        | Dispatched on               | `event.detail` |
|---------------------|---------------------------------------------|-----------------------------|----------------|
| `user-focus:active` | When the user becomes active on the page    | the controller root element | -              |
| `user-focus:away`   | When the page gets minimized or loses focus | the controller root element | -              |

## Side Effects

The controller will install event listeners for:

`Window`:
- `focus`
- `blur`


`Document`:
- `visibilitychange`


## How to Use

<StimulusPlayground
  controller="UserFocusController"
  html={`<div style="display:flex;flex-direction:column;gap:0.75rem;max-width:380px;">
  <div data-controller="user-focus"
    id="focus-indicator"
    style="padding:1rem;background:#f0fdf4;border:2px solid #bbf7d0;border-radius:8px;text-align:center;font-size:0.9rem;color:#166534;font-weight:500;transition:all 0.3s;">
    Window is focused
  </div>
  <p style="font-size:0.85rem;color:#6b7280;margin:0;">
    Click outside this preview window, then back in — the indicator updates.
  </p>
</div>`}
  js={`const indicator = document.getElementById('focus-indicator');
document.getElementById('focus-indicator').addEventListener('user-focus:active', () => {
  indicator.style.background = '#f0fdf4';
  indicator.style.borderColor = '#bbf7d0';
  indicator.style.color = '#166534';
  indicator.textContent = 'Window is focused';
});
document.getElementById('focus-indicator').addEventListener('user-focus:away', () => {
  indicator.style.background = '#fef2f2';
  indicator.style.borderColor = '#fecaca';
  indicator.style.color = '#dc2626';
  indicator.textContent = 'Window lost focus';
});`}
  editorHeight={220}
/>
