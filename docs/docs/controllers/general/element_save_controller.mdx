## Purpose

Save the state of the attribute(s) of an element to LocalStorage, and restore it, at will.

## [Actions](https://stimulus.hotwire.dev/reference/actions)

| Action    | Purpose                                                              |
|-----------|----------------------------------------------------------------------|
| `save`    | Save the state of the element's specified attributes to LocalStorage |
| `restore` | Restore the element to its previous state from LocalStorage          |
| `clear`   | Clear any saved state regarding this element from LocalStorage       |

## [Targets](https://stimulus.hotwire.dev/reference/targets)

| Target    | Purpose                                                                                                   | Default                     |
|-----------|-----------------------------------------------------------------------------------------------------------|-----------------------------|
| `element` | The element to persist state from, if not the controller root element. Particularly useful if you need to | The controller root element |


## [Classes](https://stimulus.hotwire.dev/reference/classes)
<NoClasses/>

## [Values](https://stimulus.hotwire.dev/reference/values)

| Value           | Type      | Description                                                                                                                                                                                         | Default                                                                                               |
|-----------------|-----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| `id`            | `String`  | The unique ID of this element on the page. This combined with the current URL will be used as a key to store the value in LocalStorage. This allows multiple elements on the same page to all save. | The ID of the controller root element, or throw an error an ID is not found and this is not specified |
| `attributes`    | `String`  | A space separated set of DOM element attribute names, or attribute paths that [Lodash.get](https://lodash.com/docs/#get) accepts. I.e. `dataset.foo`, `className`                                   | -                                                                                                     |
| `restoreOnLoad` | `Boolean` | Whether the controller should try to restore any previous state when the controller connects                                                                                                        | true                                                                                                  |

## Events

| Event                          | When                                                                         | Dispatched on                                         | `event.detail` |
|--------------------------------|------------------------------------------------------------------------------|-------------------------------------------------------|----------------|
| `element-save:cleared`         | When the saved state is cleared, by the `clear` action                       | Either `elementTarget` or the controller root element | -              |
| `element-save:save:success`    | When the element saves successfully                                          | Either `elementTarget` or the controller root element | -              |
| `element-save:restore:success` | When the element restores values successfully                                | Either `elementTarget` or the controller root element | -              |
| `element-save:restore:empty`   | When a restore is triggered, but there is nothing in LocalStorage to restore | Either `elementTarget` or the controller root element | -              |

  controller="ElementSaveController"
  html={`<div style="display:flex;flex-direction:column;gap:1rem;max-width:420px;">
  <p style="font-size:0.9rem;color:#374151;margin:0;">
    Type in the textarea, then click <strong>Save</strong>. The value is persisted to
    LocalStorage. Click <strong>Restore</strong> to reload it, or <strong>Clear</strong>
    to remove the saved state.
  </p>
  <div data-controller="element-save"
    data-element-save-id-value="demo-notes"
    data-element-save-attributes-value="value"
    data-element-save-restore-on-load-value="true">
    <textarea
      data-element-save-target="element"
      id="demo-textarea"
      placeholder="Type your notes here..."
      style="width:100%;height:100px;padding:0.5rem;border:1px solid #d1d5db;border-radius:6px;font-size:0.9rem;resize:vertical;box-sizing:border-box;"
    ></textarea>
    <div style="display:flex;gap:0.5rem;margin-top:0.5rem;">
      <button data-action="click->element-save#save"
        style="padding:0.4rem 0.9rem;background:#6366f1;color:#fff;border:none;border-radius:6px;cursor:pointer;font-size:0.85rem;">
        Save
      </button>
      <button data-action="click->element-save#restore"
        style="padding:0.4rem 0.9rem;background:#e5e7eb;color:#374151;border:none;border-radius:6px;cursor:pointer;font-size:0.85rem;">
        Restore
      </button>
      <button data-action="click->element-save#clear"
        style="padding:0.4rem 0.9rem;background:#fee2e2;color:#dc2626;border:none;border-radius:6px;cursor:pointer;font-size:0.85rem;">
        Clear
      </button>
    </div>
  </div>
  <div id="save-log" style="font-size:0.85rem;color:#6b7280;min-height:1.5rem;"></div>
</div>`}
  js={`const ctrl = document.querySelector('[data-controller="element-save"]');
ctrl.addEventListener('element-save:save:success', () => {
  document.getElementById('save-log').textContent = 'Saved to LocalStorage.';
});
ctrl.addEventListener('element-save:restore:success', () => {
  document.getElementById('save-log').textContent = 'Restored from LocalStorage.';
});
ctrl.addEventListener('element-save:restore:empty', () => {
  document.getElementById('save-log').textContent = 'Nothing saved yet — restore skipped.';
});
ctrl.addEventListener('element-save:cleared', () => {
  document.getElementById('save-log').textContent = 'LocalStorage cleared.';
});`}
