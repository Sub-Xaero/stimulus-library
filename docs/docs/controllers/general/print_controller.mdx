## Purpose

Utility controller to trigger the page print dialogue when wired up to an event in the DOM.


## [Actions](https://stimulus.hotwire.dev/reference/actions)

| Action  | Purpose                              |
|---------|--------------------------------------|
| `print` | Trigger the Browser's print dialogue |
