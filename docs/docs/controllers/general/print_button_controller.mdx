## Purpose

Controller that adds print functionality to a link/button, to open a print dialogue when clicked.

| Action  | Purpose                              |
|---------|--------------------------------------|
| `print` | Trigger the Browser's print dialogue |

## Side Effects

Adds event listeners for `click` on the controller root element.
