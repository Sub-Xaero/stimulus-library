## Purpose

Add the ability to dismiss/remove an element from the DOM. i.e. Notifications, Flash messages. alerts


## [Actions](https://stimulus.hotwire.dev/reference/actions)

| Action  | Purpose                          |
|---------|----------------------------------|
| dismiss | Removes the element from the DOM |

## How to Use

A good example is a flash message, rather than it staying on the page forever, allowing the user to dismiss it with a little 'X' in the corner.

<StimulusPlayground
  controller="DismissableController"
  html={`<div data-controller="dismissable"
  style="background:#fee2e2;border:1px solid #fca5a5;padding:1rem;border-radius:8px;
         display:flex;justify-content:space-between;align-items:center;max-width:420px;">
  <span>Flash: Your settings were saved successfully.</span>
  <button data-action="click->dismissable#dismiss"
    style="background:none;border:none;cursor:pointer;font-size:1.5rem;line-height:1;"
    aria-label="Dismiss">×</button>
</div>
<p style="color:#6b7280;margin-top:1rem;font-size:0.9rem;">
  Click × — the element is removed from the DOM entirely.
</p>`}
  editorHeight={220}
/>


# Browser support / Necessary polyfills

Please reference https://caniuse.com/childnode-remove, you may need to polyfill `ChildNode.remove()` using https://www.npmjs.com/package/element-remove-polyfill
if you need to support IE11 or below.
