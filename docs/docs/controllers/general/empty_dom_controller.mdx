## Purpose

Mainly for use with TurboStreams or AJAX loaded content that modifies a container interactively. When interactions mutate the DOM and add/remove content, detect when the container you attach the controller to becomes empty/not empty.

You can then style the controller, or show a nice placeholder when the container is empty, rather than showing empty space.


## [Actions](https://stimulus.hotwire.dev/reference/actions)
<NoActions/>

## [Targets](https://stimulus.hotwire.dev/reference/targets)

| Target                 | Purpose                                                  | Default                     |
|------------------------|----------------------------------------------------------|-----------------------------|
| `container` (Optional) | The DOM element whose childList to observe for emptiness | the controller root element |

## [Classes](https://stimulus.hotwire.dev/reference/classes)

| Class                 | Purpose                                                             | Default |
|-----------------------|---------------------------------------------------------------------|---------|
| `empty` (Optional)    | The class to apply to the attached element when it is empty         | `-`     |
| `notEmpty` (Optional) | The class to apply to the attached element when it is **not** empty | `-`     |

## [Values](https://stimulus.hotwire.dev/reference/values)

| Value                      | Type   | Description                                                                                                      | Default                                  |
|----------------------------|--------|------------------------------------------------------------------------------------------------------------------|------------------------------------------|
| `scopeSelector` (Optional) | String | A CSS selector to pass to `querySelectorAll` to limit what elements are included in the count of empty/not-empty | All child elements of controller element |

## Events

| Event                 | When                                                        | Dispatched on          | `event.detail`                                                 |
|-----------------------|-------------------------------------------------------------|------------------------|----------------------------------------------------------------|
| `empty-dom:empty`     | When the subtree of the attached element becomes empty      | The controller element | -                                                              |
| `empty-dom:not-empty` | When the subtree of the attached element is no longer empty | The controller element | `count` - The number of matching elements that are now present |

## Side Effects
<NoSideEffects/>


## How to Use

The controller emits an event `dom:empty` when the container becomes empty, and `dom:not-empty` when it is no longer empty (and for every DOM update that means it is still not empty). Use those events to hook up other Stimulus actions.
Alternatively, set a particular class when the watched container becomes empty or filled.

You can also scope what the controller watches for inside the container using a CSS selector in the `data-empty-dom-scope-selector-value` attribute.

The selector you specify will be used to `querySelectorAll` the children/subtree of the container you attach the controller too, and fire the
`dom:empty` and `dom:not-empty` events when the results of the query are empty/not empty.

<StimulusPlayground
  controller="EmptyDomController"
  html={`<style>
  .list-empty { opacity: 0.5; }
  .list-has-items { opacity: 1; }
</style>
<div style="display:flex;flex-direction:column;gap:0.75rem;max-width:380px;">
  <div
    data-controller="empty-dom"
    data-empty-dom-empty-class="list-empty"
    data-empty-dom-not-empty-class="list-has-items"
    style="min-height:80px;border:1px dashed #d1d5db;border-radius:8px;padding:0.75rem;transition:opacity 0.2s;">
    <div id="items-list"></div>
  </div>
  <div id="status-msg" style="padding:0.5rem 0.75rem;background:#fef9c3;border:1px solid #fde047;border-radius:6px;font-size:0.85rem;color:#713f12;">
    Container is empty — empty-dom:empty event fired
  </div>
  <div style="display:flex;gap:0.5rem;">
    <button id="add-btn"
      style="padding:0.4rem 0.9rem;background:#6366f1;color:#fff;border:none;border-radius:6px;cursor:pointer;font-size:0.9rem;">
      + Add item
    </button>
    <button id="remove-btn"
      style="padding:0.4rem 0.9rem;background:#e5e7eb;color:#374151;border:none;border-radius:6px;cursor:pointer;font-size:0.9rem;">
      Remove last
    </button>
  </div>
</div>`}
  js={`let count = 0;
const container = document.querySelector('[data-controller="empty-dom"]');

container.addEventListener('empty-dom:empty', () => {
  document.getElementById('status-msg').textContent = 'Container is empty — empty-dom:empty event fired';
  document.getElementById('status-msg').style.background = '#fef9c3';
  document.getElementById('status-msg').style.borderColor = '#fde047';
  document.getElementById('status-msg').style.color = '#713f12';
});
container.addEventListener('empty-dom:not-empty', (e) => {
  document.getElementById('status-msg').textContent = 'Container has ' + e.detail.count + ' item(s) — empty-dom:not-empty event fired';
  document.getElementById('status-msg').style.background = '#f0fdf4';
  document.getElementById('status-msg').style.borderColor = '#bbf7d0';
  document.getElementById('status-msg').style.color = '#166534';
});

document.getElementById('add-btn').addEventListener('click', () => {
  count++;
  const item = document.createElement('div');
  item.style.cssText = 'padding:0.35rem 0.5rem;background:#eef2ff;border-radius:4px;margin-bottom:0.25rem;font-size:0.9rem;color:#374151;';
  item.textContent = 'Item ' + count;
  document.getElementById('items-list').appendChild(item);
});
document.getElementById('remove-btn').addEventListener('click', () => {
  const list = document.getElementById('items-list');
  if (list.lastElementChild) list.removeChild(list.lastElementChild);
});`}
  editorHeight={280}
/>
